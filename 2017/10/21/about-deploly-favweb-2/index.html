nginx

安装nginx之前，需要先将nginx的依赖安装完：

  • gcc 安装：yum install gcc-c++
  • PCRE pcre-devel 安装：yum install -y pcre pcre-devel
  • zlib 安装: yum install -y zlib zlib-devel
  • OpenSSL 安装: yum install -y openssl openssl-devel
